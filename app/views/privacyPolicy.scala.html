3. Purpose of Data Processing

The information collected in server logs is used for the following limited purposes:

  • Ensuring the proper functioning and security of the Website.
  • Basic analysis of website traffic to understand usage patterns and improve performance.

This data is generally not used to identify individual users in a personally identifiable way.

4. Data Retention

Server logs are typically retained according to our server configuration for a period of 30 days. After this period, the logs are usually deleted or anonymized.

Given the nature of our application and the limited data we process, we do not currently share any personal data with third parties.
