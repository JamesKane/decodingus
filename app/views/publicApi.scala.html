Public API Documentation

GET /api/v1/references/details

Retrieve details for references

Returns a list of reference details, including publication information, ENA studies, and sample counts.

Responses

200 Successful response

Content Type: application/json

                        
[
    {
        "publication": {
            "url": "https://pmc.ncbi.nlm.nih.gov/articles/PMC155271/",
            "journal": "Genome Research",
            "abstractSummary": "The Y chromosome contains the largest ...",
            "id": 1,
            "pubmedId": "11827954",
            "title": "A Nomenclature System for the Tree of Human Y-Chromosomal Binary Haplogroups",
            "doi": "10.1101/gr.217602",
            "publicationDate": "2012-02-01",
            "authors": "The Y Chromosome Consortium"
        },
        "enaStudies": [],
        "sampleCount": 0
    }
]

GET /api/v1/references/details/{publicationId}/biosamples

Retrieve list of biosamples used in the publication

Returns a list of biosamples with haplogroup and location data, if available.

Responses

200 Successful response

Content Type: application/json

                        
[
    {
        "yDnaHaplogroup": "H1",
        "geoCoord": {
            "lat": 45,
            "lon": 90
        },
        "sampleName": "foo",
        "mtDnaHaplogroup": "tbd",
        "enaAccession": "ERS123456"
    },
    {
        "sampleName": "bar",
        "enaAccession": "ERS789012",
        "yDnaHaplogroup": "R2",
        "mtDnaHaplogroup": "tbd"
    }
]
}
